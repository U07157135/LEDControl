## 如何啟動
```
git clone https://github.com/U07157135/RPI_LED_CONTROL.git
cd RPI_LED_CONTROL/web
sudo python main.py
```
## 網路設定
把host改成本地端ip4的位置之後同個網段下都可以透過https://host:port/\"來連線
![](https://i.imgur.com/OgmW9ec.png)
